# 🛍️ LoomHaus — Modern Fashion eCommerce UI

A sleek, high-performance fashion eCommerce frontend built using **React + Vite**, designed with a premium, minimal aesthetic and smooth user experience.

---

## ✨ Features

* ⚡ Lightning-fast performance with Vite
* 🎨 Modern luxury UI design
* 📱 Fully responsive (mobile + desktop)
* 🛒 Interactive shopping cart system
* 🧭 Category-based navigation (Men, Women, Gen Z, etc.)
* 🔥 Smooth animations & transitions
* 💾 LocalStorage-based cart persistence
* 🧩 Component-based scalable architecture

---

## 🧱 Tech Stack

* **React (Hooks)**
* **Vite**
* **JavaScript (ES6+)**
* **CSS (Custom Styling + Animations)**

## 🚀 Getting Started

### 1. Clone the repository

```
git clone https://github.com/yourusername/loomhaus-ecommerce-ui.git
cd loomhaus-ecommerce-ui
```

---

### 2. Install dependencies

```
npm install
```

---

### 3. Run the development server

```
npm run dev
```

---

### 4. Build for production

```
npm run build
```
